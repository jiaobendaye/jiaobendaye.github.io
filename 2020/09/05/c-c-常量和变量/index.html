c/c++ 常量和变量

发表于 2020-09-05

一，const 修饰的变量

  • int * p2 = &c_b;该句在使用gcc编译时，会警告warning: initialization discards ‘const’ qualifier from pointer target type [-Wdiscarded-qualifiers]。

  • 使用clang编译时，会警告 warning: initializing 'int *' with an expression of type 'const int *' discards qualifiers [-Wincompatible-pointer-types-discards-qualifiers]。

  • clang编译后运行的结果与gcc不同。如果使用clang编译，c_b的值没有改变，仍然是10；而使用gcc编译，该值则变为20。

从警告的内容可以看出，clang的提示要比gcc更加清晰明了；从执行结果可以看出，clang实现的const更加符合const的语义。

二，字符串常量和变量

  • char * c_p = "hello world";该语句定义的指针指向常量区地址，无法修改。

  • char v_p[] = "hello world";该语句会拷贝一份字符串到栈区，v_p是指向该地址的指针。

修改字符串常量是未定义行为，不同的编译器结果可能不同，本文使用的是clang和gcc
